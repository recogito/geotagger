# Recogito GeoTagger Plugin

A geo-tagging plugin for Recogito. Work in progress.

### Installation

- Go to the folder where your `recogito-client` is installed.
- In the folder, run `npm run install-plugin recogito/geotagger`.
- Restart Recogito.
